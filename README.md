Este código corresponde à versão original de um trabalho desenvolvido para o primeiro semestre de ADS na PUCPR. Tem o foco em um sistema completo de gerenciamento acadêmico.
Ele permite o cadastro e controle de estudantes, professores, disciplinas, turmas e matrículas, utilizando menus interativos no terminal.

Por se tratar da primeira versão do projeto, o código possui uma estrutura mais extensa e detalhada, totalizando cerca de 380 linhas, com várias funções específicas para cada tipo de cadastro, além de salvamento e carregamento de dados em arquivos JSON.
